# 🧴 Состав косметики — Telegram Mini App

Анализатор безопасности ингредиентов косметики в формате Telegram Mini App.

## Что умеет

- Принимает состав косметики (через запятую или построчно)
- Классифицирует каждый ингредиент: **Вредный / Осторожно / Безопасный / Неизвестный**
- Показывает описание и конкретную причину опасности
- Адаптируется к светлой/тёмной теме Telegram
- Работает с INCI-названиями и русскими наименованиями

## База данных

~60 ингредиентов в трёх категориях:
- ⛔ **Вредные**: парабены, сульфаты (SLS/SLES), формальдегид и его высвободители, нефтепродукты, триклозан, фталаты, гидрохинон, толуол
- ⚠️ **Осторожно**: ПЭГ-соединения, BHA/BHT, силиконы, синтетические отдушки, химические UV-фильтры, соли алюминия, феноксиэтанол
- ✅ **Безопасные**: вода, глицерин, алоэ, витамины (C, E, B3, B5), гиалуроновая кислота, натуральные масла, керамиды, физические UV-фильтры

## Как запустить (деплой + подключение к Telegram)

### 1. Разместить файлы на хостинге

Нужен **HTTPS** — Telegram не принимает HTTP.

**Бесплатные варианты:**

**GitHub Pages** (рекомендуется):
```bash
# Создать репозиторий на GitHub, залить файлы
git init
git add .
git commit -m "init"
git remote add origin https://github.com/YOUR_USER/cosmetic-bot.git
git push -u origin main
# Включить Pages: Settings → Pages → Source: main branch
# URL будет: https://YOUR_USER.github.io/cosmetic-bot/
```

**Netlify** (самый простой):
1. Зайти на [netlify.com](https://netlify.com)
2. Перетащить папку `cosmetic-analyzer-bot` в браузер
3. Получить URL вида `https://xxxxxx.netlify.app`

**Vercel**:
```bash
npm i -g vercel
vercel --prod
```

---

### 2. Создать бота в Telegram

1. Открыть [@BotFather](https://t.me/BotFather)
2. Написать `/newbot` → задать имя и username
3. Сохранить полученный токен

---

### 3. Подключить Mini App к боту

В [@BotFather](https://t.me/BotFather):
```
/newapp
```
Выбрать бота → указать название → указать URL хостинга (из шага 1).

**Или через кнопку в боте** (проще):
```
/mybots → выбрать бота → Bot Settings → Menu Button → Edit menu button URL
```
Указать URL вашего хостинга.

---

### 4. Проверить

Открыть бота в Telegram → нажать кнопку меню (или `/start`) → откроется Mini App.

## Структура файлов

```
cosmetic-analyzer-bot/
├── index.html       # Главная страница Mini App
├── css/
│   └── style.css    # Стили (адаптируется к теме Telegram)
└── js/
    ├── db.js        # База данных ингредиентов (~60 штук)
    └── app.js       # Логика: парсинг, поиск совпадений, рендер
```

## Расширение базы данных

Добавить ингредиент в `js/db.js`:

```js
{
  id: 'my_ingredient',
  names: ['inci name', 'russian name', 'alias'],
  category: 'harmful', // или 'caution', 'safe'
  group: 'Группа',
  concern: 'Краткая причина опасности',
  description: 'Подробное описание почему это плохо/хорошо.'
},
```
